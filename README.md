# 계산기 프로그램

**1. 프로젝트 소개**

목적: 자바 문법 실습과 객체 지향 개념 적용

개발 환경: IntelliJ



**2. 프로젝트 단계 및 기능 설명**

Lv 1 : 클래스 없이 계산기 구현

• 두 개의 숫자와 사칙연산 문자를 입력받아 계산

• exit 입력 시까지 반복 실행 및 결과 출력

Lv 2 : 객체 지향 개념 적용 (클래스와 연산 메소드 작성)

• 계산 결과를 저장하는 컬렉션 생성

• 가장 오래된 결과를 삭제하는 기능 추가

Lv 3 :입력값 확장 (정수에서 실수로 변경)

• 저장된 결과 컬렉션 조회 기능 추가

• 특정 값보다 큰 결과만 출력하는 필터링 기능 구현



**3. 주요 기능**

• 입력 처리 및 계산

• 결과 저장 및 관리

• 결과 필터링 및 조회
